### Progression Scheme

**Block 1 (Weeks 1-3):** Linear progression, conservative
- Start at RPE 6-7 (should feel like "I could do 3-4 more reps")
- Add 2.5kg/week on barbell compounds (squat, deadlift, row, press)
- Add 1kg/week (or +1 rep) on dumbbell and accessory work
- If RPE hits 8 before week 3, hold weight and add reps instead
- Week 4: deload (50% volume, same weight)

**Block 2 (Weeks 5-7):** Continued linear with autoregulation
- Resume at Week 3 weights + 2.5kg
- RPE target: 7-8 (should feel like "I could do 2-3 more reps")
- If a set goes above RPE 8, drop weight 5% for remaining sets
- Track estimated 1RM weekly for: squat, deadlift, bench press, overhead press, row
- Week 8: deload + assessment

**When to stall/reset:**
- If weight can't increase for 2 consecutive weeks on a lift → hold for 1 more week
- If still stalled → drop 10%, increase reps to 12, rebuild over 3 weeks
- Flag for Opus review if 3+ lifts stall simultaneously (likely systemic fatigue, not programming)

## Recovery Guidelines (HRV + Sleep Based)

### Daily Decision Matrix

| HRV Status | Sleep | Recommendation |
|------------|-------|----------------|
| BALANCED + above weekly avg | >7h | Full send. Train as programmed, push RPE 7-8. |
| BALANCED + within range | 6-7h | Train as planned. Moderate RPE 6-7. |
| BALANCED | <6h | Reduce volume 30%. Drop RPE by 1. Consider swapping to mobility. |
| UNBALANCED | Any | Train if sleep was >6h, but cap RPE at 6. Otherwise, rest or mobility only. |
| LOW | Any | Rest day. Mobility or easy walk only. Do not train. |
| LOW for 2+ consecutive days | Any | Extended rest. Skip gym sessions until HRV returns to BALANCED. Flag for review if >3 days. |
| **Subjective/multi-signal override** | Any | If body battery <30, OR training readiness <40, OR user reports fatigue/soreness/illness → override the above and swap to rest or mobility regardless of HRV/sleep. When 3+ signals are degraded simultaneously (HRV + sleep + body battery + subjective), force rest day — KB rule #13. |

### Sleep Priority Protocol

**Current state:** 6.8h average, 23% of nights <6h. This is the single biggest performance lever.

**Targets:**
- **Minimum:** 7h average over any 7-day window
- **Optimal:** 7.5-8h during heavy training weeks
- **Critical threshold:** Flag any week with >2 nights below 6h

**Interventions (friendly reminders, not commands):**
- Consistent bedtime (flag if Garmin sleep_start varies >90min across the week)
- No intense training within 3h of bedtime
- On short-sleep days: reduce next-day training intensity, don't skip — light movement aids recovery
- When sleep is trending below target: "Heads up — you've got Strength A tomorrow and your sleep average this week is 6.2h. Might be worth prioritizing an early night."

### Deload Protocol

- **Scheduled:** Every 4th week (Week 4, Week 8)
- **Reactive:** Triggered if HRV LOW for 3+ days, or RPE inflation >1 point at matched loads
- **Method:** Same exercises, same weight, 50% fewer sets. Keep 1 mountain day. Add extra mobility.
- **Duration:** 5-7 days
